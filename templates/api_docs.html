⚠️ 限流说明

所有API接口都受到限流保护：每分钟最多100个请求。超过限制将返回429状态码。

搜索案例

GET /api/search

请求参数

q string (可选) 搜索关键词
offset number (可选) 偏移量，默认0
limit number (可选) 返回结果数量，默认20，最大100

请求示例

搜索包含"合同纠纷"的案例

GET /api/search?q=合同纠纷&limit=10

获取案例详情

GET /api/case/{id}

路径参数

id number (必需) 案例ID

GET /api/case/12345

错误响应格式

{
  "error": "ERROR_CODE",
  "message": "错误描述信息"
}

常见错误码

NOT_FOUND 请求的资源不存在
RATE_LIMIT_EXCEEDED 请求频率超过限制
INVALID_PARAMETER 请求参数无效
