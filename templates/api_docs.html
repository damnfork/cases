🔐 认证与限流说明

API Token 认证：所有API接口支持基于token的认证，不同token有不同的限流配置。

  • 无token：每分钟100个请求（默认限制）
  • 有效token：根据token配置的限流策略
  • 无效token：返回401 Unauthorized

Token传递方式：

  • Authorization头：Authorization: Bearer your-token-here
  • 自定义头：X-API-Token: your-token-here

搜索案例

GET /api/search

请求参数

q string (可选) 搜索关键词
offset number (可选) 偏移量，默认0
limit number (可选) 返回结果数量，默认20，最大100

请求示例

无认证搜索

curl "http://localhost:8081/api/search?q=合同纠纷&limit=10"

使用 Bearer Token 认证

curl -H "Authorization: Bearer demo-token-1" "http://localhost:8081/api/search?q=合同纠纷&limit=10"

使用自定义头认证

curl -H "X-API-Token: demo-token-1" "http://localhost:8081/api/search?q=合同纠纷&limit=10"
